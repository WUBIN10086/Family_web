# Family_web
家庭网站 - 私人使用

## 📖 简介
这是一个温馨的家庭照片展示网站，用于记录和分享美好的家庭时光。

## 🚀 快速开始

### 本地开发
1. 克隆仓库到本地
```bash
git clone https://github.com/WUBIN10086/Family_web.git
cd Family_web
```

2. 直接在浏览器中打开 `index.html` 即可预览

### 部署到 GitHub Pages（推荐）

**最简单的方式 - 无需云服务器！**

1. **启用 GitHub Pages**
   - 进入 GitHub 仓库：https://github.com/WUBIN10086/Family_web
   - 点击 `Settings`（设置）
   - 在左侧菜单找到 `Pages`
   - 在 `Source` 部分，选择 `Deploy from a branch`
   - 选择分支：`main`
   - 选择文件夹：`/ (root)`
   - 点击 `Save`

2. **自动部署**
   - 完成上述设置后，GitHub 会自动部署你的网站
   - 几分钟后，你的网站将在以下地址可用：
     ```
     https://WUBIN10086.github.io/Family_web/
     ```

3. **更新网站**
   - 只需在本地修改文件
   - 提交并推送到 GitHub：
     ```bash
     git add .
     git commit -m "更新内容"
     git push origin main
     ```
   - GitHub Pages 会自动重新部署（通常需要 1-2 分钟）
